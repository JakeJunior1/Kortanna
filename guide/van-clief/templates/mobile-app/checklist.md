# Mobile App — Deliverables Checklist

## Design

- [ ] Screen map with all screens listed
- [ ] Navigation flow diagram (stack/tab/drawer)
- [ ] State shape defined

## Components

- [ ] Design tokens (colors, typography, spacing)
- [ ] Button (primary, secondary, disabled)
- [ ] Input (text, password, validation states)
- [ ] Card component
- [ ] Loading / skeleton states
- [ ] Error states

## API Layer

- [ ] Typed API client
- [ ] Auth token stored in secure keychain
- [ ] Error handling (network, 4xx, 5xx)
- [ ] Offline detection

## Screens

- [ ] All screens implemented per screen map
- [ ] Navigation wired (back, tab, deep link)
- [ ] Loading and error states on all data screens

## Build

- [ ] iOS build config (bundle ID, app icon, splash)
- [ ] Android build config (package name, icons, splash)
- [ ] Environment variables for dev/staging/prod
- [ ] Test build runs on simulator/emulator

## Quality

- [ ] Tested on iOS (simulator, 375pt width)
- [ ] Tested on Android (emulator, 360dp width)
- [ ] No console errors or warnings in dev build
