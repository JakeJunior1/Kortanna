# Landing Page — Deliverables Checklist

## Copy

- [ ] Hero: headline + subheadline
- [ ] 3–5 value propositions with supporting detail
- [ ] Social proof (testimonials, logos, stats)
- [ ] FAQ (5+ questions)
- [ ] Primary CTA (above fold + bottom)

## Design

- [ ] Color palette (primary, secondary, neutral, background)
- [ ] Typography scale (headings, body, captions)
- [ ] Component styles (buttons, cards, badges)

## Build

- [ ] Hero section
- [ ] Features / value props section
- [ ] Social proof section
- [ ] Pricing section (if applicable)
- [ ] FAQ section
- [ ] Footer with nav links
- [ ] Mobile responsive (tested at 375px, 768px, 1280px)

## SEO

- [ ] Page title + meta description
- [ ] Open Graph tags (og:title, og:description, og:image)
- [ ] sitemap.xml
- [ ] robots.txt
- [ ] JSON-LD structured data

## Deploy

- [ ] Vercel project created
- [ ] Custom domain configured
- [ ] SSL active
- [ ] Analytics installed
- [ ] Smoke test: page loads, CTA works, form submits
